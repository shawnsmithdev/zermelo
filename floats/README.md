zermelo/floats
==============
This subpackage handles sorting float slices. 

Example
-------

```go
package main

import (
	"github.com/shawnsmithdev/zermelo/v2/floats"
	"something"
)

func main() {
	var x []float64
	x = something.GetFloatData()
	floats.SortFloats(x)
}
```
